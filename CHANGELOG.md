# Changelog

All notable changes to OpsAgent will be documented in this file.

## [1.0.0] - 2025-01-31

### Added
- Initial release
- System metrics collection (CPU, memory, disk, network)
- Configurable threshold-based alerting
- AI agent integration via OpenCode Zen (kimi-k2.5)
- Automated remediation for safe actions
- Discord notifications for human escalation
- Real-time web dashboard
- Multi-server support with Turso database
- PM2 daemon mode
- Systemd service support
- Docker deployment
- Bun runtime support (no Node.js required)
